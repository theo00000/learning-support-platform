# Learning Support Platform

Learning Support Platform is a fullstack web application designed to help high school students access learning materials for exam preparation.

## Problem

Many students need a simple and organized platform to access learning materials based on subjects and difficulty levels.

## Solution

This platform provides authentication, a personalized dashboard, subject-based filtering, material detail pages, and structured learning content.

## Features

- Student registration and login
- JWT-based authentication
- Protected dashboard
- Learning material list
- Search and subject filtering
- Material detail page
- Related materials

## Tech Stack

Frontend:

- React
- Vite
- Tailwind CSS
- Axios
- React Router

Backend:

- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs

## Architecture

Frontend -> REST API -> Express Server -> MongoDB

## How to Run

### Backend

```bash
cd Back-end
npm install
npm run dev
```
