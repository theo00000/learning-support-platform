# Learning Support Platform Case Study

## Background
High school students often need a simple way to access structured learning materials during exam preparation.

## Problem
Learning resources are scattered, and students need a faster way to find materials by subject, difficulty, and available study time.

## Target Users
High school students preparing for exams.

## My Role
Fullstack Developer  
I designed the web frontend, built the backend API, connected MongoDB, implemented JWT authentication, created the mobile version using Expo, and deployed the project using Vercel.

## Solution
Learning Support Platform provides a centralized dashboard where students can register, login, search materials, filter by subject, and view detailed learning content.

## Engineering Decisions
- Used Express.js for REST API structure
- Used MongoDB and Mongoose for flexible learning material data
- Used JWT authentication for protected routes
- Used React + Vite for fast web frontend development
- Used Expo React Native for mobile app support
- Separated backend, frontend, and mobile folders to keep the project maintainable

## Challenges
- Connecting mobile app to local backend
- Handling CORS between Vercel frontend and backend
- Managing environment variables for development and production
- Keeping data fields consistent across backend, web, and mobile

## What I Learned
I learned how to build and deploy a fullstack product, debug real production issues, manage authentication flow, and structure a project for maintainability.
